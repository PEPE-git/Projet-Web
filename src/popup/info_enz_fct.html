Les enzymes ont différentes activités chimiques. Vous pouvez effectuer des recherches basées sur ces activités.

Lorsque vous entrez votre requête, des suggestions apparaissent. Ce sont les valeurs présentes dans la base de données qui correspondent à la recherche que vous entrez. Vous pouvez choisir d'en sélectionner une ou d'entrer le mot de votre choix. Toutes les occurences de l'expression renseignée seront alors recherchées.

Attention : si vous mettez plusieurs mots clés, l'expression exacte sera recherchée. Si vous souhaitez trouver toutes les informations relatives à plusieurs mots clés, il vous faudra réaliser plusieurs recherches simples ou une recherche avancée.
