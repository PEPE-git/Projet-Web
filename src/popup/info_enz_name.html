Les noms d'enzyme se divisent en trois catégories : noms officiels (systematic names), noms communs (accepted names) et synonymes (synonyms).
La recherche peut être faite sur une des trois catégories ou sur toutes en cochant la case appropriée.

Lorsque vous tapez un nom, des suggestions s'affichent. Vous pouvez en sélectionner une ou laisser un nom plus générique (par ex. "alcohol"). Tous les noms contenant l'expression renseignée seront alors recherchés.

Attention : si vous mettez plusieurs mots clés, l'expression exacte sera recherchée. Si vous souhaitez trouver toutes les informations concernant plusieurs mots clés, il vous faudra réaliser une recherche avancée.
